# 📚 Электронная библиотека

Автоматизированная информационная система «Электронная библиотека» — веб-приложение для ведения реестра книг, поиска, оставления рецензий и формирования личных подборок.

**Группа:** 241-327  
**Студент:** Прошутин Максим Дмитриевич

---

## Стек технологий

- **Backend:** Python 3, Flask
- **БД:** MySQL + SQLAlchemy ORM
- **Миграции:** Flask-Migrate
- **Фронтенд:** Bootstrap 5, Bootstrap Icons
- **Markdown:** EasyMDE (редактор) + python-markdown (рендер)
- **Безопасность:** Bleach (санитайзер), Werkzeug (хэширование паролей)

---

## Функциональность

- Каталог книг с пагинацией (сортировка по году)
- Добавление, редактирование и удаление книг (с обложками)
- MD5-дедупликация загружаемых обложек
- Markdown-описания книг и рецензий
- Система ролей: Администратор, Модератор, Пользователь
- Регистрация и аутентификация пользователей
- Рецензии с оценками
- **Вариант 2:** Личные подборки книг

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ТВОЙ_НИК/electronic-library.git
cd electronic-library
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Создать базу данных

```sql
CREATE DATABASE library_db CHARACTER SET utf8mb4;
```

### 4. Настроить окружение

Скопировать `.env.example` в `.env` и заполнить:

```
SECRET_KEY=your-secret-key
DATABASE_URL=mysql+pymysql://root:password@localhost/library_db
```

### 5. Применить миграции

```bash
flask db init
flask db migrate -m "init"
flask db upgrade
```

### 6. Заполнить начальные данные

```bash
python seed.py
```

### 7. Запустить

```bash
python run.py
```

Приложение доступно по адресу: `http://localhost:5000`
